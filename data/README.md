# Dataset

## Geographic Bias Image Dataset
- **6,914 images** across 6 geographic regions: Africa, Americas, Europe, East Asia, South East Asia, West Asia
- **Vehicle categories:** bicycle, boat, bus, car, truck
- **Total size:** ~7.6GB
- **Access:** [Google Drive](https://drive.google.com/drive/folders/1cAepiA4AHpkyNOpbH0Nx3U1uMiKySnV4?usp=sharing)

## Structure
Images are named with the convention: `{Country}_{vehicle_type}_{id}.jpg`
Example: `Nigeria_bicycle_22602.jpg`, `Japan_bus_47695.jpg`

## Download Instructions
Download the folder from the Drive link above and place contents 
in this `data/` directory before running the notebook.
